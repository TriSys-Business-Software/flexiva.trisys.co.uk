This dialogue is used to configure the properties of both components and fields. A component can be a sophisicated user-interface item such as a grid, calendar, organogram etc... A field is bound to a data source and is used to display and capture data. Use the Apply button to update the underlying component/field and use the Preview button in the form designer to see how it looks and feels.
