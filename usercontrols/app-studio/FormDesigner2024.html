  Form Properties

  Edit  

Form properties can be managed using the edit button, or by using App Studio where forms can be created, edited and deleted.

  Tabs

  Add     Edit     Up     Down     Delete  

Tabs are used to group panels containing components and fields into logical groups. Each tab can have its own icon, caption and colour. Typically, data entry forms use a master/detail layout with tabs for each detail section.

  Components

Components can be either standard or custom allowing customers to build their own. A component is a combination of HTML, CSS and JavaScript which can contains other components, or SDK controls. At run-time (including preview) these underlying controls are rendered. Example components are labels, panels, spinners, date widgets, grids, iframes etc..

  Fields

These are the ReST API fields for the READ data source specified for this form. Only fields not already on the form are displayed here. When the field is dropped onto the form, it disappears from this list. Both of these mechanisms prevent the designer creating duplicate fields.
