All your data sources are listed here. Data sources are typically third party ReST API's which you consume in this application via data entry forms.
Selecting a data source, folder or request will display the properties in the properties panel to the right where they can be edited or deleted. Requests can also be tested to pull/push data from/to the source.
Use the Add button menu to add a new data source, folder or request. Choose import to choose a popular data source, and use the delete button to delete the selected data source, folder or request.
