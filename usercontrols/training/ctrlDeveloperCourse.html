This training course is designed specifically for software developers.

In order to gain the most from the course, you should have some knowledge of web programming technologies, specifically HTML, CSS, Javascript, JQuery. It will also be very useful to understand the system administration aspects of TriSys.

It is recommended that you watch the videos in order during the course, however you will be able to revisit this course at any time and watch any video as desired. It is also recommended that you have a computer with a fast internet connection and sound enabled so that you can enjoy high quality video broadcasting accompanied by the presenters voice.

The course programme is as follows:

  1. Introduction
    1. Overview of Developer Studio
      Summary of the capabilities of developer studio to build custom forms, code and data.
    2. Overview of Web API
      Summary of the Web Application Programming Interface and how this can be used to build compelling line of business, data-drive, cloud and mobile applications.
