  Components

Components can be either standard or custom allowing customers to build their own. A component is a combination of HTML, CSS and JavaScript which can contains other components, or SDK controls. At run-time (including preview) these underlying controls are rendered. Example components are labels, panels, spinners, date widgets, grids etc..

  Fields

These are (mainly config) fields from the database for the entity associated with this form. Only fields not already on the form are displayed here. When the field is dropped onto the form, it disappaers this list. Both of these prevent the designer creating duplicate fields.
