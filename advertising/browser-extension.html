The TriSys Web Browser Extension enables a seamless integration between your TriSys recruitment database and LinkedIn.

Key Functionality

  • Import LinkedIn contacts with a single click
  • Associate TriSys contact records with LinkedIn profiles
  • View detailed TriSys contact information inside LinkedIn
  • Run TriSys actions directly inside LinkedIn
  • Selectively update TriSys contacts with LinkedIn information
  • Automatically create companies based on LinkedIn contacts
  • Seamless window transition between LinkedIn and TriSys
  • Easy installation experience ( simply add from Chrome Web Store)
  • And much more. No LinkedIn subscription required.
  • Fully compatible with the latest TriSys CRM product versions
