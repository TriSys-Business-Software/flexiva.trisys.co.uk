Smooth curved borders are now becoming standard on both desktops, tablets and phones. TriSys Apex has now followed this trend, adding a pleasing aesthetic to the recruiting experience for consultants.

As of today, we are pleased to confirm that TriSys Apex now features a smoother user experience thanks to a comprehensive project which has resulted in the rounding of all user interface components, namely:

  • All input fields (text, numeric and more)
  • All dropdown fields (search field selection, conditional clause selection and more)
  • All pop-up windows (actions, prompts, file upload dialogues and more)
  • All forms (CV Auto Recognition, Contact, Company, Requirement, Placement, Timesheet)
  • Individual areas within forms (for example "Personal Details", "Employment Details" and more)
  • Data grids (lookup fields, pop-up column selectors and more)
  • Checkboxes on data grids (replaced square checkboxes with circular checkboxes)
  • Buttons (all buttons in TriSys Apex already had round corners prior to the latest release however we have added further roundness)

Please get in touch to learn more about this new functionality.
