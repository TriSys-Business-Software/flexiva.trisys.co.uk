Data Import
Upload your data from another system as a Microsoft Excel file and migrate it into TriSys to create multiple database records of any type.
Self-service data import of your own data is at your own risk. If in doubt, please speak with your TriSys Account Manager. Once you have imported data, this data cannot be deleted in bulk.

Step 1 of 4: Download Template File

In order to ensure an accurate and efficient data migration process, we have defined a number of Microsoft Excel formated template files which must be downloaded onto your computer, and populated with your data, retaining the same layout including column names.

  Download Excel Template
  • a. Contact and Company Template (Recommended)
  • b. LinkedIn People Profiles Template
  • c. LinkedIn Sales Navigator Profiles Template
  • d. LinkedIn Deep Profile Template (education/work histories)
  • e. LinkedIn Companies Template

Step 2 of 4: Upload File

After copying your data into this Microsoft Excel file, please ensure that data is stored in the correct columns exactly as laid out in the above template file. Once you have added your data and saved your file, please upload it using the button below:

  Upload Excel Data File

Please note that you may upload only a single file at a time, but can repeat this process numerous times in order to gradually import all of your data.
You will be prompted to confirm the number of records, and commence the import (Step 3 of 4).
After import, you will be prompted to view and verify the imported data (Step 4 of 4).
