App Store
Apps, plug-ins, extensions, API's and 3rd party products and services which augment TriSys' products and services.

Microsoft Office

TriSys Office Add-In

This component is installed as an Outlook task pane and allows recruiters to view existing TriSys contact/company details for the sender or recipients of selected e-mail messages. The add-in allows data to be easily imported into TriSys and for many workflow actions to be orchestrated.

Browser Extensions

TriSys

Designed for both Google Chrome, and latest Microsoft Chromium Edge web browsers. When you visit a social network such as LinkedIn, clicking the extension button shows the associated contact details in TriSys, allowing recruiters to import or link profiles including e-mail addresses, phone numbers, photos, job titles and much more. The extension makes your recruitment database and social networks work together to save time.

Web Portals

Website Builder

Our website builder allows you to build and maintain your new website using our integrated content management system (CMS). We also supply a range of 'widgets' which are fully integrated with your TriSys recruitment database allowing your candidates to search for live vacancies, apply for jobs, setup job alerts, complete on-line timesheets and much more.

TriSys Web Jobs

Designed for recruitment agencies to allow candidates to search and apply to live vacancies. This micro-site can be hosted and btanded inside any web site to provide a seamless candidate oriented jobs experience. Please click here to see an example of how this works in a parallax styled recruitment agency web site. Candidates can register, login, contact us, request password, search, favourite, send to friends, apply, view placements and enter timesheets.

Mobile Apps

TriSys Mobile for Android

Designed for any manufacturer running the Android operating system e.g. Samsung, Lenovo, Huawei, LG, Xiaomi, ZTE etc.. Access your client and candidate contact/company information from your phone. Make phone calls, send SMS text messages, send e-mail, open maps with directions to meetings.

TriSys Apex for Android

The Apex recruitment CRM can be installed from the play store onto your android phone with full functionality. All manufacturers running the android operating system are fully supported.

TriSys Mobile for iOS

Designed for Apple iPhone and iPad devices. Access your client and candidate contact/company information from your phone. Make phone calls, send SMS text messages, send e-mail, open maps with directions to meetings.

TriSys Apex for iOS

The Apex recruitment CRM can be installed from the play store onto your iPhone and iPad devices with full functionality.

Browser Apps

TriSys Apex Recruitment CRM for Google Chrome

Designed for the Google Chrome web browser. TriSys Apex is a cost-effective cross-platform recruitment CRM solution designed specifically for recruitment agencies. This installer creates a desktop version of the web application with the same functionality.

TriSys Recruitment CRM for Windows Store

Designed for the Windows Store. TriSys Apex is a cost-effective cross-platform recruitment CRM solution designed specifically for recruitment agencies. This installer creates a windows desktop version of the web application with the same functionality.

TriSys Web API

TriSys Web API

Designed for web developers to build sophisticated web sites for recruitment agencies to allow full candidate and client functionality to interact in real-time with the TriSys recruitment agency database. This API is platform agnostic so that developers can build both client and server side applications for any operating system. It fully supports any programming language and library with the ability to call HTTPS based web services e.g. Javascript, C#, Java, VB.Net, Ruby, Python, C++, PHP etc..

Job Broadcasting

Broadbean

Recruiters can post jobs to multiple job boards from TriSys actions. The broadbean service is a job broadcaster which is able to post to thousands of job boards.

SMS Text Messaging

24X

Recruiters can send SMS text messages directly to candidate's phone from TriSys actions. The 24X service is a relaying service which provides cost effective delivery, monitoring and analytics.

Social Networks

Buffer

By using the free buffer social aggregator, you can associate all your social networks into a single account. TriSys can then use this service to securely post jobs and alerts to one or more of your social networks.

Twitter

TriSys can prepare and merge jobs into posts sent as tweets. Candidates are then able to click on the embedded link and apply to the job via the TriSys Web Jobs service.

Facebook

TriSys can prepare and merge jobs into posts sent to your facebook company/jobs page. Candidates are then able to click on the embedded link and apply to the job via the TriSys Web Jobs service.

LinkedIn

TriSys can prepare and merge jobs into posts sent to your linkedin company/jobs page. Candidates are then able to click on the embedded link and apply to the job via the TriSys Web Jobs service.
